## 🚀 Installation & Start
### Voraussetzungen
- [Node.js](https://nodejs.org/) (v20 oder höher)
- [npm](https://www.npmjs.com/)

### Schritte
1. **Repository clonen**
   ```bash
   git clone https://github.com/BlazeR-28/web-dev-learning.git
   ```
2. **Abhängigkeiten installieren**
   ```bash
   npm install
   ```
3. **Entwicklungsserver starten**
   ```bash
   npm run dev
   ```
